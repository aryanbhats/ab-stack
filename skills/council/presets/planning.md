## Consensus Rules
Each persona evaluates the plan through their lens independently. The user acts as judge — they weigh tradeoffs and make the final call. The assistant facilitates by structuring outputs and highlighting tensions, but does not decide.

## Output Format
VERDICT: Proceed / Revise / Rethink

KEY CONCERNS: [ranked by severity]
1. ...

BLIND SPOTS: [things the plan didn't consider]
- ...

RECOMMENDED CHANGES: [specific adjustments]
- ...

DISSENTING VIEW: [strongest argument against this plan]

## Auto-Suggest Triggers
- When exiting plan mode
- When "is this the right approach?" is asked
- When a plan has been drafted but not yet validated
